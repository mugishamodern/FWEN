# FWEN Mobile

FWEN Mobile is a React Native application for managing parcel delivery and tracking.

## Project Structure

See the folder hierarchy in `/src` for detailed information about components, screens, services, and utilities.

## Getting Started

1. Install dependencies: `npm install`
2. Run on Android: `npm run android`
3. Run on iOS: `npm run ios`

## Features

- User authentication (Customer, Agent, Admin)
- Parcel creation and tracking
- QR code scanning
- Real-time updates
- Payment processing
- Analytics dashboard
